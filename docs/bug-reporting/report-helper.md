# Bug reporting helper

## Questions

1. **What is the problem?**
    * Title
    * Summary
2. **What exactly happened?**
    * Steps to reproduce
    * Actual behavior
3. **What was I doing before the problem occurred? Was I doing something else at the same time?**
    * Context of the issue
    * Preconditions
4. **Has the problem occurred multiple times?**
    * Frequency of occurrence
    * Reproducibility
5. **Do I have any screenshots or videos related to the issue?**
    * Visual evidence
    * Attachments
6. **Did I use any specific data or inputs?**
    * Test data
    * Input parameters
7. **Why do I consider this a problem?**
    * Impact on user experience
    * Expected behavior
8. **What version and where did I encounter the problem?**
    * Software version
    * Environment (e.g., operating system, browser)

## Investigation

1. **Is the problem already reported?**
    * Check existing bug reports
    * Search for similar issues
2. **Can the problem be reproduced? What are the steps to reproduce it?**
    * Document the steps clearly
    * Verify reproducibility
3. **Does the problem occur with variations of inputs? Does it occur on another platform?**
    * Test with different data
4. **Does it manifest with any special settings? Did I change any settings?**
    * Check for configuration issues
5. **Does the problem manifest in other versions of the product?**
    * Test with different versions
6. **Do I know how the behavior could be improved?**
    * Suggest potential solutions
7. **What is the impact of the problem? Does it affect other parts?**
    * Assess the severity
    * Identify any related issues
